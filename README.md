## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/rkymtc/CukurovaLemonadeProject.git
```

Proje dizinine gidin

```bash
  cd CukurovaLemonadeProject
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm start
```

  
## Kullanılan Teknolojiler

- HTML
- CSS, Bootstrap
